# 技术选择
1、后端

- 核心框架：Spring Framework 4.1
- 视图框架：Spring MVC 4.1
- 持久层框架：MyBatis 3.2
- 任务调度：Spring + Quartz 2.2.3
- 分页插件：Mybatis-PageHelper
- 数据库：MySql 5.6
- 数据库连接池：Alibaba Druid 1.0
- 日志管理：SLF4J 1.7、Log4j

2、前端

- JS框架：jQuery 1.9
- CSS框架：Twitter Bootstrap 3.3.1
- 客户端验证：Bootstrap-Validator
- 数据表格：Bootstrap-table
- 日期控件： My97DatePicker
- 弹层组件：layer2.4
- 数据可视化：echarts 3
- 编辑器：markdown
